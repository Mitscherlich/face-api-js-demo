# face-api.js demo

## Quick start

```bash
$ git clone https://github.com/Mitscherlich/face-api-js-demo.git
$ cd face-api-js-demo
$ pnpm i && pnpm dev
```

Then [http://localhost:4000] will open automatically in your default browser.
